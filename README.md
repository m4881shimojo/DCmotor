# DCmotor

機械ー電気AnalogyでのDC-モータのモデル化を行います.今回は、永久磁石型（PMDC)についてです

LTspiceではschematic (.asc) ファイルを置いておきます

pythonでは　1. Step&BODE programと

　　　　　　2. 電気回路の伝達関数を求める Sympyプログラムを置いてあります

簡単なプログラムです。検証はすべて終了してはいません。
